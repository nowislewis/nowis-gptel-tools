* Installation

1. Clone this repository:
   #+BEGIN_SRC shell
   git clone https://github.com/yourusername/gptel-tools.git
   #+END_SRC

2. Add to your Emacs configuration:
   #+BEGIN_SRC emacs-lisp
   (add-to-list 'load-path "~/path/to/gptel-tools")
   (require 'gptel-tools)
   #+END_SRC

* Usage

Basic usage with custom tools:

#+BEGIN_SRC emacs-lisp
;; Start gptel with custom tools
(gptel tools:gptel-tools)
